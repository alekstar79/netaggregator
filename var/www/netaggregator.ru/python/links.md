### Create a virtual environment (venv)

#### Create a project directory
````
mkdir app
````
##### Go to the project directory
````
cd app
````
##### Create myenv environment
````
python3 -m venv myenv
````
##### Activate the environment
````
source myenv/bin/activate
````
